# Smart Coaster Hardware Design

This repository contains the design for the Smart Coaster project. The firmware for the device
can be found at https://github.com/paulhampson/smart-coaster-fw

This design was completed with EasyEDA and an editable export is provided. For ease of preview 
a PDF export of the schematic and PCB is also provided as are a Gerber pack.
